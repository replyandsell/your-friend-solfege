# Ваш друг - СОЛЬФЕДЖИО

Интерактивный учебник по сольфеджио с уроками, аудио-упражнениями и понятной навигацией.

## Быстрый запуск локально

```bash
npm install
npm run dev
```

## Как развернуть в интернете

### Вариант 1: Vercel — самый простой
1. Загрузите проект в GitHub.
2. Откройте https://vercel.com и подключите репозиторий.
3. Выберите папку проекта и нажмите Deploy.
4. Vercel сам соберёт и опубликует сайт.

### Вариант 2: GitHub Pages
1. Загрузите проект в GitHub.
2. В настройках репозитория откройте Pages.
3. Включите GitHub Actions как источник публикации.
4. После пуша в main сайт будет опубликован автоматически.
